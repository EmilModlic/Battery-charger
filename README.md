# Single Battery Charger PCB

Custom PCB for charging a single rechargable cells through ANY USB port with integrated protection and power regulation.

Design Goals:

- Reliable charger with USB input
- Clean PCB layout
- Proper power and ground layout management
- Good thermal performance

## Overview

Features:

- Single cell charging
- Any USB input port 
- USB power negotiation with both CC and DATA pins
- Selectable fast charging levels
- Status indication LEDs
- Protection circuitry
- Compact and clean PCB layout
- USB OTG with one solder bridge

Status:

- PCB manufactured and tested
- All systems functional
